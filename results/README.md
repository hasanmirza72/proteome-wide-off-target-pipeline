# Results

These are the small result tables reported in the thesis, provided so the numbers can be
inspected without re-running the full pipeline.

- dataset_final_nonredundant.csv   (23 benchmark targets)
- self_recovery_summary.tsv        (FoldDisco self-recovery)
- offtarget_hits_high.tsv          (58 high-confidence candidates)
- offtarget_hits_bioannotated.tsv  (annotated with ChEMBL/family)
- phase5_recall.tsv                (documented off-target recall)
- phase5_enrichment_null.tsv       (enrichment vs null)
- phase5_pathways.tsv              (GO/KEGG/Reactome)
- blast_hits.tsv, foldseek_hits.tsv (baseline comparison)

Large intermediate data (structures, indexes, caches) is excluded — see the main README.
